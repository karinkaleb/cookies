# Cookie Shop
Сайт предназначен для ассортимента домашнего печенья, удобного выбора товаров и оформления заказа.
Интуитивно понятный интерфейс с возможностью просмотра каталога, добавления товаров в корзину.

🔗 **Демо:** [karinkaleb.github.io/cookies/](karinkaleb.github.io/cookies/)

## Стек технологий
### Frontend
- Angular
- TypeScript
- CSS3
- HTML5
